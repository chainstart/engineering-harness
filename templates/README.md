# Templates

The first release stores built-in profile templates in `engineering_harness.profiles`. This
directory is reserved for future file-based templates, copied docs, and executor prompts.

Frontend roadmap generation is currently template-light and stack-neutral. Use
`engh frontend-tasks` to propose dashboard, submission-review, multi-role, API-only, or CLI-only
tasks from the roadmap `experience` plan without copying a framework-specific starter.
